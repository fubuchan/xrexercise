# XRエクササイズ プロトタイプ

## 使い方

### 1. ファイルをサーバーに上げる
GitHub Pages または Netlify に index.html / signal.svg / exit.svg をアップロード。
ローカルでは動かないのでhttpsのURLが必要。

### 2. マーカーを印刷する
以下のURLからマーカーを印刷してください。

- **歩行者信号マーカー（Hiro）**:
  https://raw.githack.com/AR-js-org/AR.js/master/data/images/hiro.png

- **非常口マーカー（Kanji）**:
  https://raw.githack.com/AR-js-org/AR.js/master/data/images/kanji.png

### 3. スマホでアクセス
スマホのブラウザでURLを開き、カメラを各マーカーに向ける。

- Hiroマーカー → 歩行者信号の人がストレッチ＋「立ちっぱで疲れるわ！」
- Kanjiマーカー → 非常口の人がストレッチ＋「ちょっとストレッチしていこ！」

## Netlifyへのデプロイ手順
1. netlify.com にアクセス
2. アカウント作成
3. 「Add new site」→「Deploy manually」
4. 3ファイルをドラッグ＆ドロップ
5. URLが発行されるのでスマホでアクセス

## 注意
- HTTPSが必須（カメラアクセスのため）
- スマホのブラウザはChrome推奨
- 明るい場所でマーカーを使用してください
